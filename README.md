# Google Maps simple App

## Is used:

Google Maps API

## Getting Started

```bash
<projects>: git clone https://github.com/Zhark10/mvvm_simple_flutter.git <your-project-name> && cd <your-project-name>
<your-project-name>: pub get
```

and

Invoke Debug > Start Debugging
